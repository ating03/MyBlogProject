P Y T H O N

Python is a general-purpose programming language that is widely used, interpreted, object-oriented, and high-level programming language with dynamic semantics.

HISTORY OF PYTHON

It was created by Guido Van Rossum, and first released on February 20, 1991.

The Python programming language originally comes from Monty Python's Flying Circus, an old BBC sketch comedy show.

One of the most astonishing aspects about Python is that it was created by a single person. New programming languages are usually developed and published by huge corporations with a large number of employees, and due to copyright laws, it is difficult to identify any of the individuals involved in the project. Python is an outlier. Van Rossum did not, of course, create and evolve all of the Python components himself. The rapid spread of Python around the world is due to the tireless efforts of thousands of (often anonymous) programmers, testers, users (many of whom are not IT specialists), and enthusiasts, but it must be noted that the very first idea (the seed from which Python sprouted) came from one person – Guido.

In 1999, Guido van Rossum defined his goals for Python: an easy and intuitive language just as powerful as those of the major competitors; open source, so anyone can contribute to its development; code that is as understandable as plain English; suitable for everyday tasks, allowing for short development times.

After almost 20 years, it is evident that all of these goals have been realized. According to some sources, Python is the world's third most popular programming language, while others believe it is the fifth.

PYTHON: APPLICATIONS & SYSTEM

  • Web and Internet development (e.g., Django and Pyramid frameworks, Flask and Bottle micro-frameworks)
  • Scientific and numeric computing (e.g., SciPy – a collection of packages for the purposes of mathematics, science, and engineering; Ipython – an interactive shell that features editing and recording of work sessions)
  • Education (it’s a brilliant language for teaching programming!)
  • Desktop GUIs (e.g., wxWidgets, Kivy, Qt)
  • Software Development (build control, management, and testing – Scons, Buildbot, Apache Gump, Roundup, Trac)
  • Business applications (ERP and e-commerce systems – Odoo, Tryton)
  • Games (e.g., Battlefield series, Sid Meier\’s Civilization IV…), websites and services (e.g., Dropbox, UBER, Pinterest, BuzzFeed…)

PROGRAMMING PARADIGM IN PYTHON

Python supports three types of Programming paradigms.

  • Object Oriented programming paradigms
  • Procedure Oriented programming paradigms
  • Functional programming paradigms

Objects are the most important part of the object-oriented programming paradigm. Objects are just instances of a class that contain data members as well as method functions. Furthermore, the object-oriented programming paradigm connects data members and methods functions to support encapsulation, and the code can be easily reused thanks to the concept of inheritance. However, the major disadvantage of the object-oriented programming paradigm is that if the code is not written properly, the program becomes a monster.
